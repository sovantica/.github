# Sovantica

**Cognitive infrastructure for AI agents.**

We build the substrate AI agents run on — memory, reasoning scaffolds, verifiable state. Starting with the hardest part: memory that doesn't collapse under context-window limits, and that an agent can actually reason over.

## What we ship

**[Engrava](https://github.com/sovantica/engrava)** — memory that dreams.

An embedded cognitive graph database for AI agents. Graph memory (7 edge types), hybrid FTS5 + vector search, algorithmic dreaming (4-signal consolidation, 3 gates), tamper-evident audit trail. No Neo4j. No Redis. Just SQLite.
