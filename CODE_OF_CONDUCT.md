# Code of Conduct

The Sovantica organization and all its repositories adopt the **Contributor Covenant, version 2.1** as our Code of Conduct.

The full text is maintained upstream and available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A plain-text copy is also available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt

## Scope

This Code of Conduct applies to all spaces managed by the Sovantica organization, including:

- All repositories under [github.com/sovantica](https://github.com/sovantica) (issues, pull requests, discussions, code reviews).
- Official communication channels operated by Sovantica (email, social accounts, community forums when established).
- In-person or virtual events representing the project.

It also applies when an individual is officially representing the community in public spaces.

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the project maintainers at:

**[hello@sovantica.ai](mailto:hello@sovantica.ai)**

All complaints will be reviewed and investigated promptly and fairly. All maintainers are obligated to respect the privacy and security of the reporter.

## Enforcement

We follow the **Community Impact Guidelines** published alongside the Contributor Covenant 2.1, available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines

These outline four levels of response — from a private written warning through to a permanent ban — applied proportionally to the impact of the behavior.

For questions about this Code of Conduct that are not reports of misconduct, use GitHub Discussions in any Sovantica repository, or email [hello@sovantica.ai](mailto:hello@sovantica.ai).
